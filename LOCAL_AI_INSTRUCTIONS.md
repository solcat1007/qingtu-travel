## API端点 (localhost:3001/api/)
- /cities → 所有城市列表
- /cities/:id → 城市详情
- /cities/:id/bus → 城市公交线路 (西安有效)
- /cities/:id/metro → 城市地铁线路
- /cities/:id/attractions → 城市景点
- /attractions/:id → 景点详情
- /search?q=xxx → 搜索
- /metro-lines → 所有地铁线路 (313条)
- /metro-stations?city_id=X → 某城市地铁站点
- /bus-lines → 所有公交线路 (20条)

## 坐标转换 (经纬度 → Three.js)
```
const COORD_SCALE = 1.8;
function geoTo3D(lat, lng) {
  return {
    x: (lng - 105) * COORD_SCALE,
    y: 0,
    z: (36 - lat) * COORD_SCALE
  };
}
```
