# 晴途3D地图交互方案

## 目标
将晴途旅游网站入口改为 **3D中国地图**，支持：
1. 3D中国地图展示（可旋转、缩放）
2. 飞机航行图动画（城市间弧线动画）
3. 点击城市进入城市详情
4. 城市详情页包含公交/地铁站3D动画

## 技术选型
- **Three.js**：3D渲染引擎
- **d3-geo**：地理坐标投影
- **中国地图GeoJSON**：https://geo.datav.aliyun.com/areas_v3/bound/100000_full.json
- **Tween.js**：补间动画（航线动画）

## 文件结构
```
public/
├── index.html          → 重定向到 3d-map.html
├── 3d-map.html        → 3D中国地图入口
├── city.html          → 城市详情页（含3D交通动画）
├── js/
│   ├── 3d-map.js     → 3D地图主逻辑
│   ├── flight-path.js → 航线动画
│   └── city-3d.js    → 城市交通3D动画
└── css/
    └── 3d-map.css    → 3D地图样式
```

## 实现步骤

### Step 1: 3D中国地图基础
- 下载中国地图GeoJSON
- 用Three.js绘制3D地图（ExtrudeGeometry拉伸）
- 添加光源、控制器（OrbitControls）

### Step 2: 城市标注
- 192城市坐标转Three.js坐标
- 创建城市标注（Sprite/文字）
- 鼠标悬停高亮

### Step 3: 航线动画
- 随机生成城市间航线（弧线）
- 用Tween.js制作飞机沿弧线运动
- 航线发光效果（ShaderMaterial）

### Step 4: 城市详情页
- 点击城市标注 → 跳转 city.html?cityId=xxx
- city.html 包含：
  - 城市基本信息
  - 景点列表
  - **3D公交线路动画**（线路拉伸+站点标记+动画）
  - **3D地铁线路动画**

### Step 5: 公交/地铁3D动画
- 用Three.js绘制城市平面图
- 公交线路：折线 + 站点标记（立方体）
- 地铁线路：不同颜色折线 + 站点标记
- 动画：小车沿线路移动

## 时间安排
1. 3D地图基础 (2h)
2. 城市标注+交互 (1h)
3. 航线动画 (2h)
4. 城市详情页+3D交通动画 (3h)

总计：约8小时

## 备选方案（如果Three.js太复杂）
- 使用 **ECharts GL** 的3D地图（配置简单）
- 使用 **Mapbox GL JS** 的3D地形（需要token）
